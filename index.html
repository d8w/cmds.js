CMDS web app (Beta)

cmds.js is a web app of the collective mind data server(CMDS). It integrates nicely with OZONE Widget Framework (OWF).

Accessing the demo website

Step 1. forward port 8443 (Linux/Mac)

ssh -L8443:compute-0-0.int.autonlab.org:8443 <your_auton_account>@lop1.autonlab.org

Windows users can use PuTTY.

Step 2. point your browser to https://localhost:8443/owf, log in with the following credential

testAdmin1
password

Accept the browser security warning.
